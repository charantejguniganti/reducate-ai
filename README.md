# JG University - Ultra-Premium Redesign

A futuristic, high-performance, and ultra-premium redesign of the JG University landing page. Built with Next.js 15, Framer Motion, and GSAP for an Awwwards-level experience.

## 🚀 Live Demo
**[https://reducate-ai.vercel.app](https://reducate-ai.vercel.app)**

## ✨ Key Features
- **Cinematic UI**: Glassmorphic components with dynamic gradients and global noise filters.
- **Interactive Academics**: Tabbed school navigator with deep-dive curriculum details.
- **Campus Gallery**: Auto-playing photo slider with smooth horizontal transitions.
- **Leadership Profiles**: Premium executive cards for university leadership.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports.
- **High Performance**: Built with Next.js App Router and optimized for SEO.

## 🛠️ Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, GSAP
- **Icons**: Lucide React
- **Smooth Scrolling**: Lenis

## 📦 Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

## 🏗️ Production Build
```bash
npm run build
```
